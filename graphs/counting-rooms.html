problem 01

Counting Rooms

Problem Statement

1 ≤ n, m ≤ 1000

You are given a map of a building, and your task is to count the number of its rooms. The size of the map is n×m and each square is either floor (.) or wall (#). You can move left, right, up and down through the floor squares. Input: The first input line has two integers n and m — the height and width of the map. Then there are n lines of m characters describing the map. Output: Print one integer — the number of rooms.

Approach

Classic flood fill. Iterate over every cell; when you find an unvisited floor cell, run BFS/DFS to mark the whole connected component as visited and increment the room counter.
